JavaScript 🚀

Od logiky po interaktivní web

Umíte HTML (kostra) a CSS (kabát). Nyní je čas dát webu mozek a svaly.

  • JavaScript oživuje statické stránky.
  • Dnes běží všude: v prohlížeči, na serverech, v mobilech i v chytrých hodinkách.
  • Začneme od základů logiky, ne od barviček na webu.
Ovládání: Šipka doprava ➡️ (nebo Mezerník) pro další slide. Šipka doleva ⬅️ pro návrat.

Náš nástroj: Vývojářská konzole

Zatím nebudeme manipulovat s webovou stránkou. Musíme pochopit, jak jazyk funguje uvnitř.

Otevřete si v prohlížeči Vývojářské nástroje (F12) a přepněte na záložku Console (Konzole).

// Toto je komentář. Prohlížeč ho ignoruje.
console.log("Ahoj světe!");

Příkaz console.log() vypíše cokoli do konzole. Je to váš radar, stetoskop i rentgen při hledání chyb (debugging).

Jak spojit JS s HTML?

Kód píšeme buď přímo do HTML, nebo lépe - do externího souboru.

<!DOCTYPE html>
<html>
<head>
    <title>Můj Web</title>
</head>
<body>
    <h1>Nadpis</h1>
    
    <!-- Script dáváme VŽDY těsně před konec body -->
    <script src="muj-skript.js"></script>
</body>
</html>

Proč na konec? Aby se nejprve načetla celá stránka a uživateli se něco zobrazilo, než se začne zpracovávat složitý kód.

Proměnné (Variables) 📦

Představte si proměnnou jako krabici s nálepkou. Do krabice něco vložíme a díky nálepce to později najdeme.

let vek = 18;       
const jmeno = "Jan"; 

// var už nepoužíváme (je to starý způsob plný chyb)
  • let použijte, pokud se hodnota bude v budoucnu měnit (např. skóre ve hře).
  • const (konstanta) použijte, pokud se hodnota už nikdy nezmění (např. datum narození).

Pravidla pro pojmenování krabic

Počítač je hloupý detailista. Záleží na každém písmenu.

  • Nesmí začínat číslem (let 1hrac je chyba).
  • Nesmí obsahovat mezery (let jmeno hrace je chyba).
  • Záleží na velikosti písmen (Vek a vek jsou dvě různé krabice!).

Doporučení (Camel Case): První slovo malé, každé další začíná velkým písmenem: pocetZivotuHrace, hlavniNadpis.

Datové typy: Co dáváme do krabic?

JavaScript musí vědět, s čím pracuje. Zda s textem, nebo s číslem.

  • Number (Číslo): let x = 42; nebo let pi = 3.14; (Desetinná místa se píší s tečkou!)
  • String (Text): let jmeno = "Karel"; (Text MUSÍ být v uvozovkách, jinak si JS myslí, že je to jméno jiné proměnné).
  • Boolean (Pravdivost): let jePlnolety = true; nebo false.

Základní komunikace s uživatelem

Než se naučíme číst data z HTML formulářů, použijeme to nejjednodušší:

// 1. Zobrazení zprávy (Vyskakovací okno)
alert("Pozor, útok viru! (Dělám si srandu)");

// 2. Dotaz na uživatele (Vrací String / Text!)
let odpoved = prompt("Jak se jmenuješ?");
console.log("Uživatel se jmenuje: " + odpoved);

// 3. Otázka Ano/Ne (Vrací Boolean)
let souhlasi = confirm("Opravdu chceš smazat disk?");

Zrádnost textu vs. čísla ⚠️

Funkce prompt() vám VŽDY vrátí text (String). I když uživatel napíše číslo 15.

let a = prompt("Zadej první číslo:"); // uživatel zadá 10
let b = prompt("Zadej druhé číslo:"); // uživatel zadá 5

let soucet = a + b; 
console.log(soucet); // Vypíše: 105 !!! Proč?

Protože sčítání textů ("10" + "5") je jednoduše slepí k sobě (tzv. konkatenace). Musíme text převést na číslo!

Převod textu na číslo

K převodu slouží funkce Number().

let zadani = prompt("Zadej věk:");
let vek = Number(zadani); // Nyní je to opravdové číslo

// Kratší zápis:
let pocetRohlíku = Number(prompt("Kolik chceš rohlíků?"));

🏋️ Cvičení 1: Kalkulačka věku

Úkol pro vás:

  1. Pomocí prompt() se zeptejte uživatele na aktuální rok. Nezapomeňte to převést na číslo!
  2. Pomocí druhého prompt() se zeptejte na rok jeho narození (taky převést).
  3. Spočítejte jeho věk (aktuální rok mínus rok narození) a uložte do proměnné.
  4. Pomocí alert() vypište zprávu: "Tvůj věk je přibližně [výsledek] let."
Pro spojování textu a proměnných použijte znak plus: "Ahoj " + jmeno

Základní matematika

JavaScript umí fungovat jako kalkulačka:

  • Sčítání: +
  • Odčítání: -
  • Násobení: *
  • Dělení: /
  • Mocnina: ** (např. 2 ** 3 je 8)

C# vs. JavaScript: Kde je int a float? 🤔

Pokud znáte jiné jazyky (třeba C#), možná jste zvyklí, že 5 / 2 = 2 (celočíselné dělení).

V JavaScriptu to tak NENÍ. JS má jen jeden typ pro čísla (Number) a všechno počítá s desetinnými místy.

let vysledek = 5 / 2;
console.log(vysledek); // Vypíše rovnou 2.5!

// Co když chci celočíselné dělení? Musím zaokrouhlit:
let celeCislo = Math.floor(5 / 2); 
console.log(celeCislo); // Vypíše 2

Tajemný operátor: Modulo %

Modulo vrací zbytek po celočíselném dělení. Je to nesmírně důležitý nástroj v programování.

let zbytek = 10 % 3; 
// Výsledek je 1. Proč? Protože 10 = 3*3 + 1.

let vysledek = 14 % 5;
// Výsledek je 4. (5*2 = 10, zbývá 4).

K čemu je to dobré? Ideální pro zjišťování sudosti/lichosti čísla. Sudé číslo dělené 2 má zbytek 0.

Rozhodování programu (If / Else) 🚦

Programy musí reagovat na situace. K tomu slouží podmínky.

let vek = 15;

if (vek >= 18) {
    console.log("Vítej, dospěláku!");
} else {
    console.log("Jsi moc mladý, přístup odepřen.");
}

Kód uvnitř složených závorek {} se vykoná jen tehdy, pokud je podmínka v kulatých závorkách () pravdivá (true).

Řetězení podmínek (Else if)

Co když máme více než dvě možnosti?

let body = 85;

if (body >= 90) {
    console.log("Máš jedničku!");
} else if (body >= 75) {
    console.log("Máš dvojku.");
} else if (body >= 50) {
    console.log("Máš trojku, s odřenýma ušima.");
} else {
    console.log("Propadas, uvidíme se v září.");
}

Počítač čte podmínky odshora. Jakmile najde první pravdivou, vykoná ji a zbytek přeskočí.

Porovnávací operátory

Jak můžeme věci porovnávat?

  • > Větší než
  • < Menší než
  • >= Větší nebo rovno
  • <= Menší nebo rovno
  • === Rovná se (Přesná shoda hodnoty i typu!)
  • !== Nerovná se

Rozdíl mezi = , == a === ⚠️

Tohle je nejčastější chyba začátečníků!

  • = (Jedno rovnítko) - Přiřazení. Dává věc do krabice. let x = 5;
  • == (Dvě rovnítka) - Volné porovnání. Porovnává jen hodnotu, nezajímá ho datový typ. 5 == "5" je Pravda (True)!
  • === (Tři rovnítka) - Přísné porovnání. Musí se rovnat hodnota i datový typ. 5 === "5" je Nepravda (False).

Pravidlo: V JavaScriptu VŽDY používejte ===. Vyhnete se hromadě chyb.

🏋️ Cvičení 2: Sudé, nebo liché?

Čas spojit matematiku a rozhodovací logiku.

  1. Načtěte od uživatele libovolné číslo.
  2. Zjistěte, zda je zadané číslo sudé (zbytek po dělení dvěma je nula).
  3. Pokud je sudé, vypište uživateli zprávu "Číslo je sudé".
  4. V opačném případě vypište zprávu "Číslo je liché".

Logické operátory (A zároveň, Nebo)

Často potřebujeme splnit více podmínek najednou.

  • && (A zároveň - AND): Obě strany musí být pravdivé.
    if (vek >= 18 && maPenize === true) { ... }
  • || (Nebo - OR): Stačí, aby platila alespoň jedna strana. (Znak se píše pravý Alt + W).
    if (den === "Sobota" || den === "Neděle") { console.log("Je víkend!"); }
  • ! (Negace - NOT): Obrátí pravdu na nepravdu a naopak.
    if (!jePlnolety) { console.log("Je nezletilý."); }

Cykly (Loops) 🔁

Představte si, že máte vypsat do konzole čísla od 1 do 1000. Budete psát tisíc řádků kódu? Ne. Použijeme cyklus.

Základní princip programování je DRY (Don't Repeat Yourself - Neopakuj se).

Cykly dělají to, že opakují určitý blok kódu pořád dokola, dokud platí nějaká podmínka.

Cyklus WHILE (Dokud)

Základní typ cyklu. "Dělej toto, dokud platí podmínka."

let pocitadlo = 1;

while (pocitadlo <= 5) {
    console.log("Číslo je: " + pocitadlo);
    pocitadlo = pocitadlo + 1; // Zásadní krok! Zvýšení hodnoty.
}
// Vypíše čísla 1, 2, 3, 4, 5

⚠️ Pozor na nekonečné cykly! Pokud zapomenete přičíst (změnit stav) uvnitř cyklu, podmínka bude platit navždy a prohlížeč "zamrzne".

Zkrácené zápisy (Inkrementace)

Místo pocitadlo = pocitadlo + 1; existují zkratky, které programátoři milují.

  • pocitadlo += 1; (Zvyš stávající hodnotu o 1)
  • pocitadlo += 5; (Zvyš stávající hodnotu o 5)
  • pocitadlo++; (Tzv. inkrementace. Zvyš přesně o 1. Úplně nejkratší zápis.)
  • pocitadlo--; (Dekrementace. Sniž o 1.)

Cyklus FOR

Nejpoužívanější cyklus. Je to vlastně while nacpaný do jednoho řádku. Používáme ho, když víme, kolikrát chceme cyklus opakovat.

// for (1. start ; 2. podmínka ; 3. krok)
for (let i = 1; i <= 10; i++) {
    console.log(i);
}

Jak to funguje:

  1. Vytvoří se proměnná i (zkratka pro index) s hodnotou 1. (Provede se jen jednou na začátku).
  2. Zkontroluje se, zda je i <= 10. Pokud ano, spustí se kód ve složených závorkách.
  3. Provede se krok i++ (zvýšení o 1).
  4. Znovu se zkontroluje podmínka (bod 2)...

🏋️ Cvičení 3: Trest po škole

  1. Napište cyklus for.
  2. Nastavte počátek, podmínku a krok cyklu tak, aby kód uvnitř proběhl přesně 50krát.
  3. V těle cyklu vypište požadovaný text do konzole.
  4. Bonus: Zařiďte, aby se před každým textem vypsalo i jeho pořadové číslo (např. "1. Už nikdy...", "2. Už nikdy...").

Pole (Arrays) - Seznamy 🥞

Co když máme ve třídě 30 studentů? Nebudeme tvořit 30 proměnných (student1, student2...). Vytvoříme Pole.

Pole je jeden seznam, který drží více hodnot pohromadě v hranatých závorkách.

let zaci = ["Petr", "Jana", "Karel", "Eva"];
let znamek = [1, 3, 2, 1, 5, 4];

Indexy (Počítáme od nuly!)

Jak vytáhneme data z pole? Pomocí jejich pořadí, tzv. Indexu.

⚠️ Programátoři začínají počítat od 0, ne od 1!

let zaci = ["Petr", "Jana", "Karel"];

console.log(zaci[0]); // Vypíše: Petr
console.log(zaci[1]); // Vypíše: Jana
console.log(zaci[2]); // Vypíše: Karel
console.log(zaci[3]); // undefined (Takový prvek tam není!)

Práce s polem (Metody)

Pole umí spoustu triků.

let nakup = ["Mléko", "Chleba"];

// .length zjistí délku (počet prvků) pole. Tady to bude číslo 2.
console.log(nakup.length); 

// .push() přidá prvek nakonec
nakup.push("Máslo"); // Pole je nyní ["Mléko", "Chleba", "Máslo"]

// .pop() odstraní a vrátí poslední prvek
nakup.pop(); // Odstraní "Máslo"

Cyklus + Pole = Dokonalý pár 💑

Toto je absolutní základ programování. Jak projít všechny prvky v poli, ať jich je tam 5, nebo milion?

let teploty = [15, 20, 18, 22];

// Začínáme od indexu 0. Jedeme, dokud nedojdeme na konec pole (length)
for (let i = 0; i < teploty.length; i++) {
    // Vezmeme prvek na aktuálním indexu 'i'
    console.log("Dnešní teplota je: " + teploty[i]);
}

Všimněte si podmínky i < teploty.length. Je to ostře menší (<), protože poslední index u pole o délce 4 je 3 (0,1,2,3).

Moderní procházení: for...of (jako foreach) ✨

Klasický for cyklus s indexem i je základ, ale psát ho pořád dokola je otrava. Pro jednoduché procházení polí existují kratší cesty.

let zaci = ["Petr", "Jana", "Karel"];

// 1. Zápis "for...of" (čte se: pro každého 'zaka' z pole 'zaci')
for (let zak of zaci) {
    console.log("Jméno: " + zak);
}

// 2. Metoda .forEach() (dělá v podstatě to samé)
zaci.forEach(function(zak) {
    console.log("Jméno: " + zak);
});

Pokud nepotřebujete znát přesnou pozici (index) prvku v poli, ale stačí vám jen jeho hodnota, for...of je nejčistší a nejrychlejší volba!

Algoritmizace: Součet prvků v poli

Jak spočítat průměrnou známku? Nejdřív je musíme sečíst. Ukážeme si myšlenkový postup počítače.

let znamky = [1, 2, 5, 2];
let soucet = 0; // Pomocná proměnná pro ukládání průběžného stavu

for (let i = 0; i < znamky.length; i++) {
    soucet = soucet + znamky[i]; // Přičteme aktuální známku do součtu
}

console.log("Celkový součet je: " + soucet); // 10
console.log("Průměr je: " + soucet / znamky.length); // 2.5

Algoritmizace: Hledání maxima 🏔️

Jak byste hledali nejvyšší kartu v balíčku? Vezmete první, a pak každou další s ní porovnáte. Pokud je nová karta větší, tu původní zahodíte a držíte si novou.

let cisla = [4, 12, 5, 22, 8];

// Předpokládáme na začátku, že maximum je hned to první číslo
let maximum = cisla[0]; 

for (let i = 1; i < cisla.length; i++) {
    if (cisla[i] > maximum) {
        // Našli jsme větší číslo! Uložíme si ho.
        maximum = cisla[i];
    }
}
console.log("Největší číslo je: " + maximum); // 22

🏋️ Cvičení 4: Hledání minima

Máte pole útrat na eshopu za poslední dny: let utraty = [450, 120, 890, 50, 300];

  1. Vytvořte si toto pole v kódu.
  2. Vytvořte proměnnou minimum a vložte do ní první prvek pole (index 0).
  3. Napište cyklus for pro projití zbytku pole.
  4. Uvnitř cyklu dejte podmínku: Pokud je aktuální prvek pole menší než dosavadní minimum, uložte tento nový prvek do proměnné minimum.
  5. Po skončení cyklu vypište nalezené minimum. Mělo by to být 50.

Funkce (Functions) ⚙️

Funkce je znovupoužitelný kus kódu. Je to jako "továrna" - zabalíte do ní určitý proces, dáte jí jméno, a pak ji jen voláte, kdykoliv potřebujete.

// 1. DEKLARACE (vytvoření továrny)
function pozdrav() {
    console.log("Ahoj, vítám tě tu!");
}

// 2. VOLÁNÍ FUNKCE (spuštění továrny)
pozdrav(); // Vypíše: Ahoj...
pozdrav(); // Vypíše to znovu

Parametry a Argumenty (Vstupy do továrny)

Funkce může přijímat data zvenčí. Těmto zástupným názvům říkáme Parametry.

// 'jmeno' a 'vek' jsou PARAMETRY
function pozdravCloveka(jmeno, vek) {
    console.log("Ahoj " + jmeno + ", je ti " + vek + " let.");
}

// Při volání tam pošleme reálná data (ARGUMENTY)
pozdravCloveka("Karle", 25);
pozdravCloveka("Jano", 30);

Return (Výstup z továrny)

Funkce často nepotřebuje něco vypisovat. Chceme po ní, aby něco spočítala a vrátila výsledek zpět, abychom s ním mohli dál pracovat.

function secti(a, b) {
    let vysledek = a + b;
    return vysledek; // Vrací hodnotu ven z funkce
    // console.log("Toto se nikdy nestane, return funkci ukončí");
}

let x = secti(10, 5); // x bude mít hodnotu 15
let y = secti(20, 2); // y bude 22

console.log(x * y); // S výsledky můžeme normálně počítat dál

  3. Uvnitř funkce spočítejte obsah (šířka * výška).
  4. Použijte klíčové slovo return pro vrácení výsledku.
  5. Zavolejte funkci s čísly 5 a 10, uložte výsledek do proměnné a tu vypište. Výsledek by měl být 50.

Objekty (Objects) - Úvod

Zatímco Pole je seznam hodnot, Objekt je balíček vlastností, které popisují jednu věc (např. jednoho uživatele, jedno auto).

Objekty se zapisují do složených závorek {} jako páry "klíč: hodnota".

let student = {
    jmeno: "Lukáš",
    vek: 20,
    studujeIT: true,
    znamky: [1, 2, 1, 3]
};

Práce s objekty

K vlastnostem objektu přistupujeme pomocí tečkové notace.

let auto = {
    znacka: "Škoda",
    model: "Octavia",
    barva: "černá"
};

console.log(auto.znacka); // Vypíše: Škoda

auto.barva = "červená"; // Změna hodnoty vlastnosti
auto.rokVyroby = 2020;  // Přidání úplně nové vlastnosti

Objekty umí i něco dělat (Metody) ⚔️

Doteď měly naše objekty jen vlastnosti (pasivní data). Ale objekt může mít i své vlastní funkce! Těm říkáme metody.

let hrac = {
    jmeno: "Artur",
    zivoty: 100,
    // Toto je metoda (funkce uložená v objektu)
    pozdrav: function() {
        console.log("Připrav se na souboj!");
    }
};

// Funkci zavoláme pomocí tečky, stejně jako když jsme přidávali do pole
hrac.pozdrav();

Objekty a změna vlastního stavu 🚗

Pomocí slova this (já) může objekt uvnitř metody měnit svá vlastní data. (Všimněte si také moderního, kratšího zápisu metody bez slova function).

let auto = {
    znacka: "bmw",
    barva: "cerna",
    rok_vyroby: 2025,
    pocet_mist: 4,
    engine_status: false,
    
    startMotoru() { // Moderní kratší zápis!
        console.log("Motor se startuje");
        this.engine_status = true;
    },
    vypnutiMotoru() {
        console.log("Motor se vypíná");
        this.engine_status = false;
    }
};

console.log("Stav motoru: " + auto.engine_status); // false
auto.startMotoru();
console.log("Stav po startu: " + auto.engine_status); // true

Herní příklad: Hráč vs. NPC 👾

Pojďme naprogramovat souboj. Vytvoříme objekty, které na sebe budou útočit. Pomocí magického slova this (znamená "tento konkrétní objekt") může objekt sahat na své vlastní vlastnosti.

let hrac = {
    jmeno: "Rytíř",
    zivoty: 100,
    sila: 15,
    zautocNa: function(cil) {
        cil.zivoty -= this.sila; // Odečteme životy cíli podle "mojí" síly
        console.log(this.jmeno + " útočí! " + cil.jmeno + " má teď " + cil.zivoty + " HP.");
    }
};

Všimněte si: cil je parametr (ten, koho jdeme praštit), this jsem já sám (ten, kdo útočí).

Souboj v aréně! 🎬

Teď vytvoříme Rytíři protivníka a necháme je, ať si to vyříkají.

let skret = {
    jmeno: "Skřet",
    zivoty: 50,
    sila: 8,
    zautocNa: function(cil) {
        cil.zivoty -= this.sila;
        console.log(this.jmeno + " útočí! " + cil.jmeno + " má teď " + cil.zivoty + " HP.");
    }
};

// Akce začíná!
hrac.zautocNa(skret);  // Vypíše: Rytíř útočí! Skřet má teď 35 HP.
skret.zautocNa(hrac);  // Vypíše: Skřet útočí! Rytíř má teď 92 HP.
hrac.zautocNa(skret);  // Vypíše: Rytíř útočí! Skřet má teď 20 HP.
Takhle přesně fungují reálné hry! Vše na obrazovce je objekt s vlastnostmi (pozice, HP) a metodami (pohyb, útok).

🏋️ Cvičení 6: Léčivý lektvar 🧪

Váš Rytíř dostal zásah a potřebuje se vyléčit.

  1. Vytvořte si v kódu objekt hrac z předchozích ukázek.
  2. Přidejte mu zbrusu novou metodu (funkci) s názvem vypijLektvar. (Nezapomeňte ji oddělit čárkou od předchozích vlastností!)
  3. Tato metoda nebude potřebovat žádný parametr.
  4. Uvnitř metody zvyšte rytířovy zivoty (musíte použít this.zivoty) o 20.
  5. Do metody přidejte výpis do konzole: "Rytíř vypil lektvar a má teď [počet] HP".
  6. Vyzkoušejte metodu zavolat: hrac.vypijLektvar();
