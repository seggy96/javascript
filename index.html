JavaScript 🚀

Od logiky po interaktivní web

Umíte HTML (kostra) a CSS (kabát). Nyní je čas dát webu mozek a svaly.

  • JavaScript oživuje statické stránky.
  • Dnes běží všude: v prohlížeči, na serverech, v mobilech i v chytrých hodinkách.
  • Začneme od základů logiky, ne od barviček na webu.
Ovládání: Šipka doprava ➡️ (nebo Mezerník) pro další slide. Šipka doleva ⬅️ pro návrat. Přejeďte myší přes pravou stranu nebo klikněte dovnitř, aby fungovaly klávesy.

Náš nástroj: Vývojářská konzole

Zatím nebudeme manipulovat s webovou stránkou. Musíme pochopit, jak jazyk funguje uvnitř.

Otevřete si v prohlížeči Vývojářské nástroje (F12) a přepněte na záložku Console (Konzole).

// Toto je komentář. Prohlížeč ho ignoruje.
console.log("Ahoj světe!");

Příkaz console.log() vypíše cokoli do konzole. Je to váš radar, stetoskop i rentgen při hledání chyb (debugging).

Konzole přímo ve VS Code? Jde to! 💻

Nechce se vám při tréninku logiky neustále překlikávat do prohlížeče? Zkuste to přímo v editoru:

  • Rozšíření Code Runner: Nainstalujte si ve VS Code doplněk "Code Runner". Pak stačí v .js souboru kliknout na ikonku "Play" ▶️ vpravo nahoře a výsledek uvidíte dole na záložce Output.
Tento doplněk extrémně urychluje učení algoritmů a trénování kódu.

Jak spojit JS s HTML?

Kód píšeme buď přímo do HTML, nebo lépe - do externího souboru.

<!DOCTYPE html>
<html>
<head>
    <title>Můj Web</title>
</head>
<body>
    <h1>Nadpis</h1>
    
    <!-- Script dáváme VŽDY těsně před konec body -->
    <script src="muj-skript.js"></script>
</body>
</html>

Proč na konec? Aby se nejprve načetla celá stránka a uživateli se něco zobrazilo, než se začne zpracovávat složitý kód.

Proměnné (Variables) 📦

Představte si proměnnou jako krabici s nálepkou. Do krabice něco vložíme a díky nálepce to později najdeme.

let vek = 18;       
const jmeno = "Jan"; 

// var už nepoužíváme (je to starý způsob plný chyb)
  • let použijte, pokud se hodnota bude v budoucnu měnit (např. skóre ve hře).
  • const (konstanta) použijte, pokud se hodnota už nikdy nezmění (např. datum narození).

Pravidla pro pojmenování proměnných

Záleží na každém písmenu.

  • Nesmí začínat číslem (let 1hrac je chyba).
  • Nesmí obsahovat mezery (let jmeno hrace je chyba).
  • Záleží na velikosti písmen (Vek a vek jsou dvě různé krabice!).

Doporučení (Camel Case): První slovo malé, každé další začíná velkým písmenem: pocetZivotuHrace, hlavniNadpis.

Datové typy: Co dáváme do krabic?

JavaScript musí vědět, s čím pracuje. Zda s textem, nebo s číslem.

  • Number (Číslo): let x = 42; nebo let pi = 3.14; (Desetinná místa se píší s tečkou!)
  • String (Text): let jmeno = "Karel"; (Text MUSÍ být v uvozovkách, jinak si JS myslí, že je to jméno jiné proměnné).
  • Boolean (Pravdivost): let jePlnolety = true; nebo false.

Základní komunikace s uživatelem

Než se naučíme číst data z HTML formulářů, použijeme to nejjednodušší:

// 1. Zobrazení zprávy (Vyskakovací okno)
alert("Pozor, útok viru! (Dělám si srandu)");

// 2. Dotaz na uživatele (Vrací String / Text!)
let odpoved = prompt("Jak se jmenuješ?");
console.log("Uživatel se jmenuje: " + odpoved);

// 3. Otázka Ano/Ne (Vrací Boolean)
let souhlasi = confirm("Opravdu chceš smazat disk?");

Zrádnost textu vs. čísla ⚠️

Funkce prompt() vám VŽDY vrátí text (String). I když uživatel napíše číslo 15.

let a = prompt("Zadej první číslo:"); // uživatel zadá 10
let b = prompt("Zadej druhé číslo:"); // uživatel zadá 5

let soucet = a + b; 
console.log(soucet); // Vypíše: 105 !!! Proč?

Protože sčítání textů ("10" + "5") je jednoduše slepí k sobě (tzv. konkatenace). Musíme text převést na číslo!

Převod textu na číslo

K převodu slouží funkce Number().

let zadani = prompt("Zadej věk:");
let vek = Number(zadani); // Nyní je to opravdové číslo

// Kratší zápis:
let pocetRohlíku = Number(prompt("Kolik chceš rohlíků?"));

🏋️ Cvičení 1: Kalkulačka věku

Úkol pro vás:

  1. Pomocí prompt() se zeptejte uživatele na aktuální rok. Nezapomeňte to převést na číslo!
  2. Pomocí druhého prompt() se zeptejte na rok jeho narození (taky převést).
  3. Spočítejte jeho věk (aktuální rok mínus rok narození) a uložte do proměnné.
  4. Pomocí alert() vypište zprávu: "Tvůj věk je přibližně [výsledek] let."
Pro spojování textu a proměnných použijte znak plus: "Ahoj " + jmeno

Základní matematika

JavaScript umí fungovat jako kalkulačka:

  • Sčítání: +
  • Odčítání: -
  • Násobení: *
  • Dělení: /
  • Mocnina: ** (např. 2 ** 3 je 8)

Tajemný operátor: Modulo %

Modulo vrací zbytek po celočíselném dělení. Je to nesmírně důležitý nástroj v programování.

let zbytek = 10 % 3; 
// Výsledek je 1. Proč? Protože 10 = 3*3 + 1.

let vysledek = 14 % 5;
// Výsledek je 4. (5*2 = 10, zbývá 4).

K čemu je to dobré? Ideální pro zjišťování sudosti/lichosti čísla. Sudé číslo dělené 2 má zbytek 0.

Rozhodování programu (If / Else) 🚦

Programy musí reagovat na situace. K tomu slouží podmínky.

let vek = 15;

if (vek >= 18) {
    console.log("Vítej, dospěláku!");
} else {
    console.log("Jsi moc mladý, přístup odepřen.");
}

Kód uvnitř složených závorek {} se vykoná jen tehdy, pokud je podmínka v kulatých závorkách () pravdivá (true).

Řetězení podmínek (Else if)

Co když máme více než dvě možnosti?

let body = 85;

if (body >= 90) {
    console.log("Máš jedničku!");
} else if (body >= 75) {
    console.log("Máš dvojku.");
} else if (body >= 50) {
    console.log("Máš trojku, s odřenýma ušima.");
} else {
    console.log("Propadas, uvidíme se v září.");
}

Rozdíl mezi = , == a === ⚠️

Tohle je nejčastější chyba začátečníků!

  • = (Jedno rovnítko) - Přiřazení. Dává věc do krabice. let x = 5;
  • == (Dvě rovnítka) - Volné porovnání. Porovnává jen hodnotu, nezajímá ho datový typ. 5 == "5" je Pravda (True)!
  • === (Tři rovnítka) - Přísné porovnání. Musí se rovnat hodnota i datový typ. 5 === "5" je Nepravda (False).

Pravidlo: V JavaScriptu VŽDY používejte ===. Vyhnete se hromadě chyb.

🏋️ Cvičení 2: Sudé, nebo liché?

Čas spojit matematiku a rozhodovací logiku.

  1. Načtěte od uživatele libovolné číslo.
  2. Zjistěte, zda je zadané číslo sudé (zbytek po dělení dvěma je nula).
  3. Pokud je sudé, vypište uživateli zprávu "Číslo je sudé".
  4. V opačném případě vypište zprávu "Číslo je liché".

Cykly (Loops) 🔁

Představte si, že máte vypsat do konzole čísla od 1 do 1000. Budete psát tisíc řádků kódu? Ne. Použijeme cyklus.

Základní princip programování je DRY (Don't Repeat Yourself - Neopakuj se).

Cykly dělají to, že opakují určitý blok kódu pořád dokola, dokud platí nějaká podmínka.

Cyklus WHILE (Dokud)

Základní typ cyklu. "Dělej toto, dokud platí podmínka."

let pocitadlo = 1;

while (pocitadlo <= 5) {
    console.log("Číslo je: " + pocitadlo);
    pocitadlo++; // Zásadní krok! Zvýšení hodnoty.
}

⚠️ Pozor na nekonečné cykly! Pokud zapomenete uvnitř cyklu počítadlo zvýšit, podmínka bude platit navždy a prohlížeč "zamrzne".

Cyklus FOR

Nejpoužívanější cyklus. Je to vlastně while nacpaný do jednoho řádku. Používáme ho, když víme, kolikrát chceme cyklus opakovat.

// for (1. start ; 2. podmínka ; 3. krok)
for (let i = 1; i <= 10; i++) {
    console.log(i);
}

🏋️ Cvičení 3: Trest po škole

Bart Simpson je po škole a musí napsat 50x na tabuli "Už nikdy nebudu házet křídou".

  1. Napište cyklus for.
  2. Nastavte počátek, podmínku a krok cyklu tak, aby kód uvnitř proběhl přesně 50krát.
  3. V těle cyklu vypište požadovaný text do konzole.
  4. Bonus: Zařiďte, aby se před každým textem vypsalo i jeho pořadové číslo (např. "1. Už nikdy...", "2. Už nikdy...").

Pole (Arrays) - Seznamy 🥞

Co když máme ve třídě 30 studentů? Nebudeme tvořit 30 proměnných (student1, student2...). Vytvoříme Pole.

Pole je jeden seznam, který drží více hodnot pohromadě v hranatých závorkách.

let zaci = ["Petr", "Jana", "Karel", "Eva"];
let znamek = [1, 3, 2, 1, 5, 4];

Indexy (Počítáme od nuly!)

Jak vytáhneme data z pole? Pomocí jejich pořadí, tzv. Indexu.

⚠️ Programátoři začínají počítat od 0, ne od 1!

let zaci = ["Petr", "Jana", "Karel"];

console.log(zaci[0]); // Vypíše: Petr
console.log(zaci[1]); // Vypíše: Jana
console.log(zaci[2]); // Vypíše: Karel
console.log(zaci[3]); // undefined (Takový prvek tam není!)

Přidávání a odebírání (Push & Pop)

Pole nejsou statická. Můžeme do nich za běhu programu přidávat nové prvky nebo je mazat. Používáme na to tzv. metody.

let kosik = ["Mléko", "Chleba"];

// 1. PUSH: Přidá nový prvek na KONEC pole
kosik.push("Máslo"); 
console.log(kosik); // ["Mléko", "Chleba", "Máslo"]

// 2. POP: Odebere POSLEDNÍ prvek z pole
let odebranaPolozka = kosik.pop(); 
console.log(kosik); // ["Mléko", "Chleba"]
console.log("Odebral jsem:", odebranaPolozka); // Odebral jsem: Máslo

Představte si to jako balíček karet. push() položí kartu na vrch, pop() z vrchu kartu vezme.

🏋️ Cvičení 4: Playlist a fronta

Vytvoříme si jednoduchou logiku přehrávače písniček.

  1. Vytvořte pole playlist se třemi názvy písniček.
  2. Uživatel si přidal novou písničku. Pomocí push() ji přidejte do pole.
  3. Vypište do konzole celkovou délku playlistu (použijte vlastnost .length).
  4. Přehrávač dohrál poslední písničku a má ji smazat ze seznamu. Použijte pop().
  5. Vypište do konzole finální podobu playlistu.

Cyklus + Pole = Dokonalý pár 💑

Toto je absolutní základ programování. Jak projít všechny prvky v poli, ať jich je tam 5, nebo milion?

let teploty = [15, 20, 18, 22];

// Začínáme od indexu 0. Jedeme, dokud nedojdeme na konec pole
for (let i = 0; i < teploty.length; i++) {
    console.log("Dnešní teplota je: " + teploty[i]);
}

Funkce (Functions) ⚙️

Funkce je znovupoužitelný kus kódu. Je to "továrna" - zabalíte do ní určitý proces, dáte jí jméno, a pak ji jen voláte.

// 1. DEKLARACE (vytvoření továrny)
function pozdravCloveka(jmeno, vek) {
    console.log("Ahoj " + jmeno + ", je ti " + vek + " let.");
}

// 2. VOLÁNÍ FUNKCE (spuštění s reálnými daty)
pozdravCloveka("Karle", 25);
pozdravCloveka("Jano", 30);

Objekty (Objects) - Úvod

Zatímco Pole je seznam hodnot, Objekt je balíček vlastností, které detailně popisují jednu věc.

Objekty se zapisují do složených závorek {} jako páry "klíč: hodnota".

let student = {
    jmeno: "Lukáš",
    vek: 20,
    studujeIT: true,
    znamky: [1, 2, 1, 3] // Objekt může obsahovat i pole!
};

Práce s objekty

K vlastnostem objektu přistupujeme nejčastěji pomocí tečkové notace.

let auto = {
    znacka: "Škoda",
    model: "Octavia",
    barva: "černá"
};

console.log(auto.znacka); // Vypíše: Škoda

// Změna hodnoty vlastnosti
auto.barva = "červená"; 

// Přidání úplně nové vlastnosti do existujícího objektu!
auto.rokVyroby = 2020;

🏋️ Cvičení 5: Herní postava

Trénink práce s objekty.

  1. Vytvořte objekt hrac. Dejte mu vlastnosti: jmeno (text), uroven (číslo 1), a inventar (pole se dvěma texty, např. "Meč", "Lektvar").
  2. Vypište do konzole pouze jméno hráče.
  3. Hráč postoupil na další úroveň. Zvyšte mu vlastnost uroven o 1.
  4. Hráč našel "Štít". Použijte push() k přidání "Štítu" do jeho inventare.
  5. Vypište celý objekt hrac do konzole pro kontrolu.

Příklady na procvičení na test 🏆

Teorii máte za sebou. Tyto úlohy slouží jako trénink na test a simulují běžné problémy z praxe (e-shopy, bankomaty, hry).

  • Nehledejte složitá řešení, používejte jen to, co jsme se naučili (if, for, pole, objekty, funkce).
  • Pokud se zaseknete, rozdělte si problém na menší kroky.

📝 Úloha 1: E-shop doprava zdarma

Píšete logiku pro nákupní košík. E-shop nabízí dopravu zdarma pro objednávky nad 1500 Kč.


Úkol: Vytvořte proměnnou cenaObjednavky a dejte jí nějakou číselnou hodnotu. Napište podmínku (if/else), která do konzole vypíše buď "Doprava je zdarma!", nebo "Do dopravy zdarma chybí [X] Kč." (kde [X] je přesně dopočítaná chybějící částka).

📝 Úloha 2: Kontrola bezpečnosti hesla

Uživatel se registruje a my musíme ověřit, jestli si nezvolil příliš krátké heslo.


Úkol: Načtěte od uživatele heslo do proměnné. Napište podmínku, která zkontroluje jeho délku (vlastnost .length funguje i na texty!). Pokud je heslo kratší než 8 znaků, vypište varování. Jinak vypište "Heslo je v pořádku".

📝 Úloha 3: Odpočet k Novému roku

Na webové stránce se má zobrazit dramatický odpočet.


Úkol: Napište cyklus (zvolte for nebo while), který vypíše do konzole čísla sestupně od 10 do 1. Jakmile cyklus skončí, vypište na závěr "Šťastný Nový rok!".

📝 Úloha 4: Hodnota košíku

V e-shopu už nemáme jen jednu částku, ale pole s cenami jednotlivých položek, které si zákazník přidal do košíku.


Úkol: Vytvořte pole ceny = [120, 50, 490, 10];. Napište cyklus, který toto pole projde, všechny ceny sečte dohromady a výsledek uloží do nové proměnné celkem. Tu nakonec vypište.

📝 Úloha 5: Filtrování dospělých

Máte seznam věků návštěvníků kina, ale film je přístupný až od 18 let. Potřebujete zjistit, kdo může dovnitř.


Úkol: Vytvořte pole věků navstevnici = [15, 22, 17, 30, 12, 18]; a druhé, prázdné pole povoleni = []. Pomocí cyklu a podmínky projděte návštěvníky, a ty, kterým je 18 a více, přidejte (push) do pole povoleni.

📝 Úloha 6: Formátování produktu (Funkce)

Když na e-shopu vypisujete stovky produktů, nechcete HTML generovat ručně. Potřebujete na to funkci.


Úkol: Napište funkci vypisProdukt(nazev, cena). Funkce nesmí nic vypisovat do konzole, ale musí přes return vrátit zformátovaný text ve tvaru: "Produkt: [nazev] - Cena: [cena] Kč". Zavolejte funkci a její výsledek uložte do proměnné (s klíčovým slovem let) a vypište.

📝 Úloha 7: Tiskárna štítků (Pole objektů)

V praxi většinou nemáte pole obyčejných čísel, ale pole objektů. Zkombinujeme obojí.


Úkol: Vytvořte pole zamestnanci. V něm budou dva objekty. Každý objekt bude mít vlastnost jmeno a pozice (např. "Programátor", "Grafik"). Pomocí cyklu projděte toto pole a vypište "Štítek: [jméno] - [pozice]".

📝 Úloha 8: Hledání nejdražší položky

Zákazník chce na e-shopu seřadit zboží nebo najít ten nejdražší kousek.


Úkol: Máte pole hodnoty = [450, 1200, 30, 890, 5000, 200];. Napište algoritmus (cyklus + podmínka), který projde celé pole a najde v něm nejvyšší číslo. To na konci vypište. (Nápověda: Na začátku si jako "dosavadní maximum" uložte první prvek pole).

📝 Úloha 9: Simulátor Bankomatu (Část 1)

Pojďme naprogramovat logiku účtu.


Úkol: Vytvořte objekt ucet, který má vlastnosti majitel a zustatek (např. 10000). Dále napište funkci vyberPenize(castka). Tato funkce zkontroluje, zda je na účtu dostatek peněz. Pokud ano, odečte částku ze zůstatku a vypište "Úspěšný výběr". Pokud ne, vypište "Nedostatek prostředků".

📝 Úloha 10: Simulátor Bankomatu (Část 2)

Bankomat si musí pamatovat historii transakcí pro výpis.


Úkol: Rozšiřte objekt ucet z předchozí úlohy o vlastnost historie, což bude prázdné pole []. Upravte funkci pro výběr tak, že pokud je výběr úspěšný, přidá (push) do pole historie text, např. "Výběr: -500". Na konci proveďte dva úspěšné výběry a vypište celou historii účtu.

Příklady EXTRA 🧠

Tyto úlohy jsou pro ty, kteří milují logické hádanky, matematiku a hledání skrytých vzorů.

  • Už to není jen o zápisu syntaxe. Je to o algoritmickém myšlení.

🧩 EXTRA 1: Ztracená zásilka

Máte pole čísel, které představuje ID přijatých balíků na skladě. Balíky měly přijít v přesném pořadí od nejmenšího po největší. Jeden ale cestou vypadl.

let zasilky = [1, 2, 3, 4, 6, 7, 8]; // Chybí 5!

Úkol: Napište cyklus, který toto pole projde a zjistí, které číslo chybí. Program musí fungovat, ať už chybí 5, nebo třeba 3. Vypište: "Chybí balík číslo: [X]".

Nápověda: Pokud vezmete aktuální prvek a přičtete k němu 1, mělo by se to rovnat následujícímu prvku v poli. Co když se to nerovná?

🧩 EXTRA 2: Zrcadlové slovo (Palindrom)

Palindrom je slovo, které se čte stejně zleva doprava i zprava doleva (např. "radar", "kajak"). S textem (Stringem) se v JS dá pracovat podobně jako s polem – můžete ho procházet cyklem.


Úkol: Napište funkci jePalindrom(slovo). Funkce projde text od zadu dopředu, poskládá z něj nové "obrácené" slovo a porovná ho s originálem. Pokud se rovnají, vraťte `true`, jinak `false`.

Nápověda k cyklu odzadu: for (let i = slovo.length - 1; i >= 0; i--)

🧩 EXTRA 3: Hackování trezoru

Máte pole s čísly (váhy závaží). Trezor se otevře pouze tehdy, když na něj položíte přesně dvě závaží, jejichž součet je roven cílové váze (např. 70).

let zavazi = [10, 40, 50, 20, 80];
let cil = 70;
